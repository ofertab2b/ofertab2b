¿Qué incluye esta demo?

Esta publicación en GitHub Pages hospeda una versión estática de la aplicación web. El contenido está orientado a la presentación y a la experiencia de usuario, pero no ejecuta consultas reales.

  • Interfaz de consulta libre con formulario de SQL.
  • Diseño para consulta por teléfono usando un placeholder :value.
  • Página de consulta de fase Starlink con diseño de formulario.
  • Compatibilidad móvil y fácil lectura en pantallas pequeñas.

Cómo usarlo en GitHub Pages

Sube el repositorio a GitHub y habilita Pages desde la rama principal. La carpeta raíz del repositorio contendrá las páginas estáticas listadas a continuación.

https://<tu_usuario>.github.io/<tu_repositorio>/
Para ejecutar consultas reales necesitas el backend Flask con conexión a Oracle. Esta versión estática es ideal para presentar el front-end y el flujo de navegación.

Vista previa de la interfaz

Los botones y formularios están deshabilitados en esta demo para indicar que se trata de una versión de presentación.
