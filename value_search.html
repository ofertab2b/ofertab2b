Una búsqueda basada en placeholders

En una aplicación real, el formulario tomaría una consulta SQL y un valor de teléfono. El backend reemplazaría el placeholder :value y devolvería los resultados correspondientes.

Descripción de la funcionalidad

Esta página explica cómo funciona la consulta por teléfono sin ejecutar realmente el backend. Ideal para mostrar al usuario final el comportamiento esperado.

SELECT * FROM TMP_CMO WHERE MSISDN = :value
Si necesitas datos reales, el backend Flask del repositorio debe ejecutarse conectado a Oracle.
