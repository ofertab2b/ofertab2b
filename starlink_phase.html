Esta página describe cómo se vería la consulta de fase Starlink en una aplicación real. El diseño es responsive y está preparado para presentación pública.

Qué muestra esta demo

La interfaz muestra un formulario para ejecutar una consulta SQL con parámetro :value, pero no se ejecuta en GitHub Pages.

SELECT * FROM starlink_phase_table WHERE phone = :value
La funcionalidad real se activa únicamente cuando el backend Flask está en ejecución y conectado a Oracle.

Vista previa del formulario

Si deseas resultados reales, ejecuta el backend Flask del proyecto conectado a la base de datos Oracle.
